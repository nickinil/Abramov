# Book Tracker — Трекер прочитанных книг

**Автор:** Абрамов Никита Валерьевич

## Описание программы

Book Tracker — это графическое приложение на Python с использованием библиотеки `tkinter`.  
Программа предназначена для управления списком прочитанных книг.

Приложение позволяет:

- добавлять книги;
- указывать название, автора, жанр и количество страниц;
- просматривать книги в таблице;
- фильтровать книги по жанру;
- фильтровать книги по количеству страниц;
- сохранять данные в JSON;
- загружать данные из JSON;
- удалять выбранные книги;
- видеть статистику по количеству книг и страниц.

## Требования

- Python 3.6 или выше;
- стандартная библиотека `tkinter`;
- дополнительные библиотеки не требуются.

1. Скачайте проект.

## Как пользоваться

### Добавление книги

1. Введите название книги.
2. Введите автора.
3. Введите жанр.
4. Введите количество страниц.
5. Нажмите кнопку **Добавить книгу**.

Пример:

```text
Название: Мастер и Маргарита
Автор: Михаил Булгаков
Жанр: Роман
Количество страниц: 480
```

После добавления книга появится в таблице и сохранится в `books.json`.

### Фильтрация по жанру

В поле **Жанр содержит** можно ввести жанр, например:

```text
Фэнтези
```

После нажатия кнопки **Применить фильтр** будут показаны только книги подходящего жанра.

### Фильтрация по страницам

В поле **Страниц больше** можно указать число, например:

```text
200
```

Программа покажет только книги, где количество страниц больше 200.

### Сохранение и загрузка

- Кнопка **Сохранить в JSON** сохраняет данные в файл `books.json`.
- Кнопка **Загрузить из JSON** загружает данные обратно в таблицу.

## Проверка ввода

Программа проверяет:

- поля не должны быть пустыми;
- количество страниц должно быть числом;
- количество страниц должно быть больше нуля.
