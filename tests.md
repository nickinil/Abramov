# Тестирование приложения Book Tracker

**Проект:** Book Tracker  
**Автор:** Абрамов Никита Валерьевич

## 1. Проверяемые функции

- запуск графического интерфейса;
- добавление книги;
- проверка пустых полей;
- проверка количества страниц;
- сохранение данных в JSON;
- загрузка данных из JSON;
- фильтрация по жанру;
- фильтрация по количеству страниц;
- удаление выбранной книги;
- отображение статистики.

## 5. Проверка JSON

Пример структуры файла `books.json`:

```json
[
  {
    "title": "Мастер и Маргарита",
    "author": "Михаил Булгаков",
    "genre": "Роман",
    "pages": 480
  }
]
```

Проверено:

- данные сохраняются в JSON;
- данные загружаются после перезапуска;
- русские символы отображаются корректно;
- после удаления записи JSON обновляется.
